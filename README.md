# 🚦 DoNotLate - 교통정보 기반 출발 시간 알림 서비스

> **절대 늦지 마세요!** 실시간 교통 상황을 분석하여 최적의 출발 시간을 계산하고, Push 알림으로 제때 알려드립니다.

🔗 **서비스 URL** : [https://donotlate.kro.kr/](https://donotlate.kro.kr/)

---

## 📌 주요 기능

### 🕐 교통정보 기반 시간 계산
- 실시간 교통 상황을 분석하여 **최적의 출발 시간** 자동 계산
- 목적지까지의 예상 소요 시간을 교통 데이터 기반으로 제공

### 🔔 Push 알림
- 계산된 최적 출발 시간에 맞춰 **Push 알림** 발송
- 놓치지 않도록 사전 알림 설정 지원

---

## 🛠️ 서브 기능

| 기능 | 설명 |
|------|------|
| 📊 교통 차트 | 시간대별 교통 혼잡도 통계 조회 |
| 🌤️ 날씨 조회 | 오늘 및 주간 날씨 정보 제공 |
| 📢 공지사항 | 서비스 업데이트 및 공지 확인 |
| 👤 마이페이지 | 내 정보 수정 및 계정 관리 |
| 🔕 알림 관리 | Push 알림 설정 및 관리 |
| 🔐 소셜 로그인 | OAuth 2.0 기반 Google / Naver / Kakao 로그인 |
